Java各个版本语法变化

版本 年份 更新内容
Java 5 2004 枚举、泛型、注解、封箱(解箱)等操作
Java 6 2006 @Override 接口
Java 7 2011 Diamond语法，多 try catch 操作
Java 8 2014 Lambda语法、可重复注解、类型注解
Java 9 2017 模块化、接口私有方法等
Java 10 2018 局部变量类型推断
